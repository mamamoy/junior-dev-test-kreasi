
# Junior Dev Test Kreasi AI

Website ini digunakan untuk mengerjakan test yang diberikan oleh Kreasi AI dengan studi kasus pembuatan crud feedback.




## Build with

- Laravel 9
- Vite
- Vue 3
- Inertia
- Tailwind
- Axios
- Sweet Alert 2
- Template by Flowbite
